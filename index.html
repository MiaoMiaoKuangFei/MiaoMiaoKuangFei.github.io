使用Hexo搭建博客

前言我一直想搭一个自己的网站，奈何自己搞个服务器也不是很必要（我们穷鬼是这样的啦），那么用Hexo搭建一个博客，将代码托管到Github就显得很划算了。 建立这个博客的初衷还是想把自己学过的和实际使用过的技术进行复盘，督促自己不要变成咸鱼，主要是把自己摔过的坑记录下来，除非十分必要，否则过程从略记录。 要搭建这个博客大概需要以下的几个方面准备： 有一台电脑。 安装Hexo。 Github账号。 MarkDown编辑博文（感觉MarkDown的语法比Latax简洁很多啊！！！）。 各种软件的安装Node安装首先得在官网上把这个Node下载下来，大概就是无脑下载和安装就可以了（找好路径之后，一直点Next，最后Finish）。 Git的安装下载Git也是大概无脑安装就行。下载完成之后去cmd里输入git version，如果出现了它的版本信息那就是没问题了。 在这个部分中，我始终没搞清楚Git的三个桌面快捷方式到底分别是干啥的，不过好在弄这个博客只用得到Git Bush。 Hexo的安装在桌面的Git Bush中使用下面的命令。 npm install hexo-cli -g npm
